Ejercicio 1

Determina cuál de las siguientes variables son válidas y explica por qué:

$_myvar, $_7var, myvar, $myvar, $var7, $_element1, $house*5

Respuesta:

  • $_myvar es válida porque inicia con guión bajo.
  • $_7var es válida porque inicia con guión bajo.
  • myvar es inválida porque no tiene el signo de dolar ($).
  • $myvar es válida porque inicia con una letra.
  • $var7 es válida porque inicia con una letra.
  • $_element1 es válida porque inicia con guión bajo.
  • $house*5 es inválida porque el símbolo * no está permitido.

Ejercicio 2

Proporcionar los valores de $a, $b, $c como sigue:
$a = "ManejadorSQL";
$b = 'MySQL';
$c = &$a;

  1. Ahora muestra el contenido de cada variable
    • ManejadorSQL
    • MySQL
    • ManejadorSQL
  2. Agrega al código actual las siguientes asignaciones:
    $a = "PHP server";
    $b = &$a;
  3. Vuelve a mostrar el contenido de cada uno
    • PHP server
    • PHP server
    • PHP server
  4. Describe y muestra en la página obtenida que ocurrió en el segundo bloque de asignaciones

    Respuesta:

    En el primer bloque de asignaciones a contaba con el valor de "ManejadorSQL", en el segundo bloque se actualiza a PHP server, $b ahora es una referencia a $a, esto quiere decir que cualquier cambio en $a se refleja también en $b. Se tiene en cuenta que $c también es una referencia a $a por lo tanto cualquier cambio en la varible $a afecta directamente a las variables con una referencia a ella
